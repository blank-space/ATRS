# ATRS
### What is ATRS?

**Algorithm**，每周至少做一道 Leetcode。主要是为了编程训练和学习。如果不训练你看再多的算法书，你依然不会做算法题。

**Review**，阅读并点评至少一篇英文文章。主要是学习英文。

**Tips**，学习至少一个技术技巧，主要是总结和归纳你在日常工作中所遇到的知识点。

**Share**，分享一篇有观点和思考的技术文章。主要是建立你的影响力，能够输出价值观。
